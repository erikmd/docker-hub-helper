# docker-hub.py

## Summary

This repo gathers a python3 program [docker-hub.py](./docker-hub.py),
to help maintain multi-branches, automated-build repos on Docker Hub.

## Installation

Put this program in the `PATH` and run: `docker-hub.py -h`
